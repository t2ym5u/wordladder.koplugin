# wordladder.koplugin

A Word Ladder plugin for [KOReader](https://github.com/koreader/koreader).

## Rules

Transform the start word into the target word by changing exactly one letter per step, with every intermediate step a valid word of the same length. Reach the target word in as few steps as possible.

## Concept

Classic word-ladder puzzles: change one word into another, one letter at a time.

## Features

- **Configurable word length**
- **Three difficulty levels** — Easy, Medium, Hard
- **Hint** — reveals a valid next word
- **Undo**
- **Win tracking** — counts solved puzzles per session
- **Auto-save** — in-progress game restored on next launch

## Controls

| Action | How |
|--------|-----|
| Submit a word | Type it and confirm |
| Request a hint | Tap **Hint** |
| Undo last word | Tap **Undo** |
| New puzzle | Tap **New puzzle** |
| Change word length / difficulty | Tap **Word length** / **Difficulty** |

## Installation

1. Download `wordladder.koplugin.zip` from the [latest release](../../releases/latest).
2. Extract into the `plugins/` folder of your KOReader data directory.
3. Restart KOReader.
4. Open the menu → **Tools** → **Word Ladder**.
